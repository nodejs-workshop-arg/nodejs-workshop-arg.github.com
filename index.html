Node.js de 0 a 100

El primer workshop de Node.js en Español. En Buenos Aires.

Queres aprender a programar aplicaciones web realtime que escalen? Escuchaste hablar de node.js y te gustaria aprender sobre esta nueva plataforma? En este workshop de un día entero vas a salir sabiendo lo necesario para desarrollar una aplicacion en node! El formato del workshop es teorico-practico: cada modulo tiene una introducción al tema y luego una serie de ejercicios asociados para poner en practica lo aprendido.

Que tenes que saber para hacer el curso?

Para sacarle provecho a todo este curso, tenes que tener un minimio de cononcimientos de Javascript y una PC con el sistema operativo de tu preferencia. Node funciona en Windows, Linux y Mac.

Quienes dictan el workshop?

Jose Romaniello

A software craftsman with ten years of experience developing applications with Microsoft tools and platforms. Agile software developer with strong skills in Test Driven Development and agile methodologies. Member and creator of various open source projects.

José has been working with node.js for the last two years developing a Backend as a Service Platform named KidoZen. He also works as a node.js consultant for some companies in Argentina.

José drives a blog in english, another in spanish and does a lot of opensource.

Matias Woloski

Matias is a software architect, craftsman and technologist. He spent the last 12 years working on numerous software projects with companies like Microsoft Corp. He’s a published author on cloud computing and identity management. He’s a founder of identity management startup Auth10, teaches cloud computing on the University and a frequent speaker at national and international events. Matias experience with node started a year ago while working on a collaborative realtime markdown editor http://markdownr.cloudapp.net. He then started a community driven blog around node and Windows to share knowledge with the community and earlier this year he was a speaker on the first latin american jsconf.
